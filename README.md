# KALT | Premium Landing Page 🌌

A high-end, editorial-style landing page built for the **KALT** brand, focusing on minimal aesthetics, sophisticated typography, and smooth motion design.

---

## 🛠 Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Database:** [Prisma](https://www.prisma.io/) (PostgreSQL/Supabase)

---

## ✨ Key Features

- **Modern Editorial UI:** Inspired by Awwwards-winning websites with a focus on white space and clean grids.
- **Smart Navigation:** Interactive navbar that hides on scroll and reappears on hover at the top.
- **Visual Depth:** Layered background architecture using `#F5F5F3` (Off-white) and `#E5E5E3` (Light Grey).
- **Fluid Animations:** Seamless transitions and micro-interactions for a premium user experience.
